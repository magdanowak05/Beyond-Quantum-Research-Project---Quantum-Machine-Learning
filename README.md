# Beyond-Quantum-Research-Project---Quantum-Machine-Learning
Repository of a research project on quantum machine learning - comparisons of methods for classifying signals that may come from exoplanets (data from the Kepler mission)
> Analyzing Transit Method Analogies: parallels from previous work during the ESA hackathon, where  allesfitter had been used to manually model light curves for targets like KELT-3b and WASP-189b
> Signal Classification Research: gathering and reviewing literature regarding various signal classification methods, focusing on comparing the performance of classical Convolutional Neural Networks (CNNs) against quantum alternatives like Variational Quantum Classifiers (VQC)
> Stage crucial for understanding how existing models handle high-dimensional astronomical time-series data
